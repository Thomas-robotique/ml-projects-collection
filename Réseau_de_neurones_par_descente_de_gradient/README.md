# Réseau de Neurones avec Descente de Gradient

Ce projet vise à créer un réseau de neurones entraîné par descente de gradient.
Dans un premier temps, le réseaux de neurones sera appliqué sur une base de données fictive afin de classer des données et de faire des prédictions sur de nouvelles entrées.
Ensuite, à partir de ce neurone, je développerai un réseau de neurones destiné à la classification d’images.


##  Objectifs

Implémenter un réseau de neurones à partir de zéro (sans frameworks comme TensorFlow ou PyTorch pour le moment).
- Comprendre le fonctionnement de la descente de gradient
- Tester l’apprentissage sur une base de données simulée
- Préparer une base pour des extensions futures (classification réelle, visualisation, etc.)

##  Technologies utilisées

- Python 3.10+
- NumPy pour les opérations matricielles
- Matplotlib pour visualiser la perte ou les prédictions

## Situation actuelle
Actuellement, j'ai développé un neurone avec l'algorithme de descente de gradient.  
Un unique neurone est suffisant pour faire de la classification de données simples.  
Cependant, pour faire de la classification d'images, je vais devoir développer un réseau de neurones entier.
